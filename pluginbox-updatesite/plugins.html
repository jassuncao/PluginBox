EasyShell Adds an option to the popup-menu in the navigator (or package browser) view which opens a shell window. The current (or working) directory of that shell is the directory that was selected in the navigator. Custom shell command configurable in preferences page. Supported platforms:
  • Windows DOS-Shell / Explorer
  • Windows Cygwin-Bash / Explorer/
  • Linux KDE Konsole / Konqueror
  • Linux Gnome Terminal / Nautilus (not tested!)
  • Linux CDE Xterm / Dtfile (not tested!)
