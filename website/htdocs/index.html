Introduction

The Pluginbox is a container for several Eclipse plugins to come. The simple reasons behing creating this project for those plugins, even if they are not related to each other, were:

  • Their sourcecode should be made available in a public CVS repository
  • Opening a sourceforge project for each single plugin would be overkill
  • There was no other fitting place for these plugins (most plugin-collections are centered around a certain theme etc.)
